Secret Detector v1.0.0
======================

Installation:
  chmod +x secret-detector-1.0.0-linux-x86_64.run
  sudo ./secret-detector-1.0.0-linux-x86_64.run

Uninstallation:
  sudo secret-detector-uninstall

Usage:
  CLI:  secret-detector --help
  GUI:  secret-detector-gui

GitHub: https://github.com/CHIKENlvl19/SecretDetector
